《远见：如何规划职业生涯3大阶段》读书笔记

Posted on 2019-08-21 Edited on 2019-08-27 In 职业生涯 Views: Valine: 6.2k

读书感悟

读这本书的原因

第二份工作满2年的时候，这个时候，发现下一步的路程非常重要，如果选择错误，那么想在职业生涯中混的不错的概率就会很低很低；恰巧在知乎上看到有人推荐这本书，便来拿看看，期望，可以解答我的困惑。

对本书的评价

相比较直接告诉你结论的书，我更喜欢告诉你思路及方法的书。在本书中，你可以看到很多工具，很多框架，看完后我的想法是，光看是不够的，最重要的是实践。

实践就需要接受自己内心的惭愧、不堪，将真实的自己呈现出来，去选择自己的职业道路。

读完本书的感受
第一，超乎我预料的，确实是，没想到职业生涯有40+之久，意味着，现在我的问题，可能在职业的长河中，只是小浪花；
第二，对职业生涯而言，有一个roadmap是一件很重要的事情，尽管可能现在的规划很粗糙，很弱智，但我们可以一直对它进行优化，进行调整。如果走一步看一步的话，会错失掉很多机会，也会在很多时候因小失大。
第三，热爱工作，热爱生活

接下来，最最重要的，就是躲在小房间里，把下面的工具自己实践一遍！



书籍内容

相关内容有些多，建议大家看的时候，结合左边的目录一起看，会更有结构性。

前沿：成长旅程中有三个定位：岗位定位（28岁前），管理定位（32岁前），行业定位（38岁前）
职业生涯到底是为了取得成功还是为了发现快乐？于本书作者而言，构建一段成功的职业生涯的目的并不仅在于找到你热爱的工作，而且是建立起你热爱的生活。

本书将分为三个部分：

  • 正确的职场思维、框架和工具
  • 实用性建议和案例，帮助你应对职业生涯的三个主要阶段
  • 以现实生活为基础，主题包括如何平衡职业策略与家庭的关系、怎样处理跨国调动和职场危机

第一部分：正确的职场思维、框架和工具

1、打造职业规划要知道的5件事：

1．职业生涯的持续时间长得惊人，包括了三个截然不同的阶段。 职业生涯的持续时间比人们想象的长得多，至少有45年之久，而且未来很可能会进一步延长。

  • 第一阶段是强势开局的时候。你在职业上的努力必须着重于为前方的漫长道路挖掘和装备自己。
  • 第二阶段是聚焦长板的时候。该阶段的首要目标是寻找自己的甜蜜区，即你所擅长的、所热爱的和这个世界所需要的这三者之间的交集。
  • 第三阶段致力于实现持续的影响力，以及寻找一条可以稳定延续到60多岁甚至70多岁的新的可持续职业道路。

2．职场燃料很重要，因为职业生涯的基础决定结果。这种燃料有三个主要来源：可迁移技能、有意义的经验和持久的关系。
3．职业生涯需要通过对时间的巧妙投资来构建。
4．职业生涯并不是以线性或者可预测的方式发展的。
5．职业生涯远不止于一份工作，而是生活的一大部分。

2、实现职业规划要做到的5件事：

1．学习职场数学，树立正确的长期思维方式。
2．盘点职场清单，梳理你最有用处的技能、经验和关系。
3．进行“100小时测试”并完成一份“个人时间档案”， 从而了解你目前的时间投资状况。
4．在尝试建立新的职场路径或者在多个选项中抉择时，运用“职场路径向导” （Career Path Navigator）。
5．时刻更新你的职业生涯。常问我如何避免被机器取代？我能在哪里以什么方式找到工作？未来我讲如何分配时间？我会把钱花光么？工作如何能让我更幸福？


3、3大阶段，聚焦45年职业生涯

职业生涯可以被分成三个主要阶段，每一阶段都持续大约15年，且各有各的主导策略。

  • 第一阶段：加添燃料，强势开局；
  • 第二阶段：聚焦长板，达到高点；
  • 第三阶段：优化长尾，持续发挥影响力。

第一阶段：加添燃料，强势开局

职业生涯前15年的唯一目标就是为接下来的两个阶段打好基础。
有耐心，不断行动，找到自己的长板和热情所在，树立良好工作习惯，储存职场燃料，探索和弥补自身短板。

第二阶段：锚定甜蜜区，聚焦长板

在你的长板、你的爱好以及这个世界的需求之间寻找交集的时候，确认甜蜜区、确实更高追求、投资职场燃料以拓展长板，以及想方设法脱颖而出的时候，专注于自己的长板，与能够帮助你、弥补你短板的人为伴。

第三阶段：优化长尾，发挥持续影响力

确定接班人，从执行或领导的角色转变成顾问或辅助的角色，保有活力和关联性。


4、储备3大职场燃料

职业生涯中存在三种基本形式的职场燃料，可迁移技能，有意义的经验，持久的关系。

1号燃料：可迁移技能

  • 解决问题的能力；
  • 说服式沟通技巧；将自己的观点以清晰、简洁的方式呈现出来都是一种基本技能。简单制胜，使用“给妈妈写信”的练习，让无关人员听得懂。
沟通障碍
- 当两个人使用不同的说法描述同一件事情时，对话就发生了
- 当两个人使用相同的说法，但指的是不同的事情时，冲突就产生了
- 当双方没有任何沟通基础时，就会产生沟通断层
  • 完成任务的能力;有能力吸引和调动尖端人才的个人领袖通常都能成功
  • 帮助和求助的能力
  • 情商（EQ）,推荐书籍：尼尔·戈尔曼的《情商3》与布拉德伯利（Bradberry）和格里夫斯（Greaves）的《成功EQ密码》（Emotional Intelligence 2.0 ）

其他作者最喜欢的可迁移技能：

  • 如何与别人进行眼神交流和握手。
  • 如何搜索信息
  • 如何呼吸。

2号燃料：有意义的经验

  • 在不同的环境中尝试不同的事情、试验不同的做事方法，这样能创造出更强的决策技能。
  • 寻找机会分别到一个大公司和创业公司的环境中工作，要找机会去国外工作或者至少去几个不同的城市，还要努力发起一些新项目、处理一次危机，并在明显存在个人失败风险的情况下筹办一场大型活动或演出。

3号燃料：持久的关系

  • 你的雇主，雇主品牌形象
  • 你的上司，影响你的工作方法&习惯，是否能够传授经验，是否业绩突出指的尊敬
  • 你的客户，是否欣赏你，不离不弃
  • 商业伙伴，是否有优秀的商业伙伴
  • 身边的人才，问自己问题：如果我自己创建公司，周围的人里，我愿意带上谁，以及他们是否会接受我的邀请？
  • 找到你的同类，通过社群找到同伴

5、5个数字，树立正确的职场思维

5个数字是哪5个：

1．职业生涯的长度：用62减去你目前的年龄。
2．精通一项技能所需的时间：要花多少小时才能在某一方面达到“精通”？
3．40岁之后能赚到的个人财富百分比：在40岁之后，你赚到的钱会占你一生个人财富的百分之多少？ 大部分人的估计是60%。年轻人倾向于估一个比较小的数字，例如40%。真正的答案其实是85%～90%。
4．社交货币：你有多少社交网络好友？ 思考这个问题的目的是，搞清楚根据手上的社交和商业关系，自己拥有多少“社交货币”。
5．职场支持者的人数：你认为能在“职业生涯的天堂”里遇到多少人，也就是说有多少人能对你的职业生涯和人生带来真正的变化？

6、4大黄金问题，评估职业价值

【三类人生导师：明星，他们是成功的行为榜样，告诉我们如何成功；贤者，他们就像苏格拉底，并不为我们提供答案，而是教我们如何思考；策动者，他们激励我们，鞭策我们，偶尔迫使我们踏出关键的一步。】

1号燃料的盘点清单

  • 学位、专业证书
  • 语言，包括音乐和计算机语言
  • 上司和同事在你的360度绩效评估中经常提到的优点
  • 关于情商的评价
  • 人才账户，你至今雇佣或提拔过的人，谁发展的更好，他们是否愿意为你工作

2号燃料的盘点清单（工作内外）

  • 个人旅行
  • 国外工作经历
  • 企业管理或创业经验
  • 社区或志愿者活动
  • 个人贡献的大型活动、产品发布或知名的项目
  • 公开演讲、写作或表演的经验
  • 教学、咨询或指导的经验
  • 兴趣爱好、业余活动和工作之余的热情所在
  • 其他生活中的经验和挑战

3号燃料的盘点清单

  1. 联系人
    • 你参加的校友会（学校或老同事组织的）
    • 其他组织或行业协会
    • 可能对你的职业生涯带来影响的其他重要联系人
  2. 专家团
    评估两方面：专家来源，咨询的问题【使用excel表格】
  3. 关键同事
    评估两方面：关键同事，关系状态（正向/中立/负向）【使用excel表格】
  4. 支持者
    写出你认为算得上或者可能成为你的支持者的人的名字。
    如果你将自己当成一个学生，那就自然会找到老师。
  5. 你自己

4大黄金问题

在制定/回顾年度目标的时候，以下问题可帮助你完成该项工作：

  • 我是否正在学习和成长？
  • 我是否正在对某些人、现在的公司，乃至整个社会拥有影响力？
  • 我体验到乐趣了吗？
  • 我是否得到了适当的奖励，并创造了经济价值？

7、100小时测试，合理投资时间

这是作者50岁的时间档案

  • 时间有限，关注时间饼图的高效能部分
  • 前摄时间迁移法，将部分生活或者工作提前或推后，短时间集中精力在某一方面
  • 碎片时间的回报

8、职场路径向导，做出正确的职业决策

要知道自己的职场路径，首先我们要先回答三个问题

  1. 你的职业理想是什么，或者至少假设一个你可能想要达到的目标。
  2. 你目前手上有什么职场燃料？
  3. 你需要什么职场燃料才能实现这个终极理想？

是否需要硕士学位
对某些领域和某些人而言，研究生学位无疑能增加些附加值。努力获取硕士学位的原因因人而异，但你需要确定它会给你的职业生涯带来什么样的特殊价值，思考它能否：
增加你目前不具备的可迁移技能？
帮助你重塑自我，改变职业生涯的方向？
建立新的人际关系，并拓展职业生态系统？
获得你目前没有的重要证书？
加速你的探索步伐，即通过实践验证自己真正擅长和热爱的东西？

权衡投资回报率

  • 快速成长才能获得长远成功。
    如果要成长得再快一点，你就需要一份满足以下条件的工作：
  • 你周围都是比你聪明的人；
  • 你有失败的机会；
  • 公司有让你这样的人肩负重大责任的传统。

在做出决策时厘清正确的思维方式。

你必须要厘清自己的理想、技能和选择，并完全确定自己对目前状况的推论、假设都是正确的；除了职位，你更需要考虑的是工作内容和学习曲线；做好承受压力和承担风险的准备。在这个过程中，感到害怕是很正常的。然后，你就可以做决定了

我们总是会低估在未来才能兑现的好处，这种现象被称为‘时间贴现’（temporal discounting）
大部分人都会有“损失厌恶”（loss aversion）
可参考文章，宾夕法尼亚大学沃顿商学院（Wharton School）的文章《他处并不一定比此处更好》（The Grass is Not Always Greener ）

第二部分：职业生涯的三个主要阶段

第一阶段：加添燃料，强势开局

找出你擅长什么、不擅长什么、喜欢做什么，以及不喜欢做什么。

给首次求职者的8个小提示

1．利用在读的时间储备早期形式的职场燃料。

  • 可迁移技能
  • 有意义的经验
  • 持久的关系

2．制订求职作战计划。
3．积极参与校园招聘。
4．高效地进行在线申请。
5．最重要的是，用好你的联系人。
6．在与联系人见面之前，做些功课。
7．但是，找到第一份工作依然难于上青天！

善意实业的求职建议
关注长坂、能力和技能
成功的求职背后都有两个关键点：成功的意愿以及联系人与支持者组成的后援团
微小的成功对于培养动力和信息能起到很大的作用

8．探索。 即使你一脚步入了职场，探索之旅也并没有终止，而是刚刚开始。你需要弄明白自己擅长什么、热爱什么，以及这个世界需要什么。

每个成功的创业者都要直面质疑

成为你自己的品牌经理

  • 了解更多公司情况，成为某方面的专家
  • 高效沟通，并选择正确的媒介
    • 首先，话题是什么？
    • 其次，写下你的三个重点，
    • 最后，直白地说出你希望听众接下来怎么做。
  • 纸巾推动，向理想的领导力中道靠拢
    “我们讨论的是这样一个领导力的指标，你现在处于这里，接下来要往这个方向发展”。我称之为“纸巾推动”（napkin nudge）
    • 建立你在社交平台和整个行业的声望
    • 找出与你的工作切实相关的6～10个因素，请别人对你的表现和风格提些反馈意见。
(纸巾推动)

理解自身的价值，为自己的贡献争取公平的奖励

  • 尽可能多地关注产出
  • 找你的上司一对一地聊一聊，最好再找至少一名导师沟通一下。问问他们，你所关注的优先事项和目标是否正确，也就是说，它们是不是对于公司及其业务有真正价值的东西，而不是细枝末节的小项目。
  • 如果有升职的计划了，那你就得想想有没有可靠的候选人能接替你现在的职位。

一杯咖啡，导师关系加速器

第二阶段：聚焦长板，达到高点

我能获得多高的成就，我该如何找到下一步的方向；我要怎么才能找到甜蜜区，同时避免倦怠和迂腐；我如何才能不靠加班和毁掉自己的生活来扩大我的影响力；我该怎样从第一阶段打下的坚实根基中收获红利？第二阶段是识别、拓展和投注你的长板的时候。第二阶段是要创造真正的差异

你要不断问自己这三个难题：我擅长什么？我爱好什么？这个世界需要什么？

精通的秘方就是意愿和时间

  • 专注度决定了学习的深度。
  • 保持一直向前的动力，就能持续成功
  • 把眼光从小圈子中解放出去，去不同的地方，去不同的行业，只要机会OK
  • 你必须自己帮自己开门

给初任管理者的6条建议：

1．你的仪容、态度和举止正受到高度的关注和广泛的效仿。
2．一旦你确定了某个愿景，就应该简洁地表达出来，并且不停地重复重复再重复。
3．早早决定让谁上船。
4．每一个有意义的商业问题都是少数人在一间安静的小会议室里解决的。 有争议的问题最好是在较小的团队中解决，而不要放到严肃的大型公开会议上讨论。
5．你要表现得像个被人信赖的解答者，而不是高高在上的老板。
6．你并不是无所不知。

改变节奏或者改变战场

在经历第二阶段时，你应当不断地细微调整，有时还要大幅修正自己的职业生涯方向。询问以下问题：

  • 学习：我是否正在积累有助于成长的新的技能、经验和关系？
  • 影响力：我是否正在改变个人、公司，甚至整个社会？
  • 乐趣：我的职业总体上算不算我生活中正能量和乐趣的来源？
  • 奖励：我是否正在积累经济价值？

顶级首席执行官需要具备的5大特质：
1．诚实和契合。
2．智力上的好奇和敏捷。
3．提升业务业绩的历史记录。
4．真实、自我意识以及平衡。
5．活力和热情。

第三阶段：优化长尾，持续发挥影响力

  • 试验，自愿接受挑战
  • 创业，开辟全新疆域
  • 管理学习曲线，在一切还没失效前行动

第三部分：Work-life Balance

如何做好职场父母

职业生涯与为人父母的共存之道

1．不要让职业生涯和为人父母成为非此即彼的选项。
2．找到一个热爱家庭的雇主。
3．如果没有后方的恰当支持，你就无法生存。
4．设立现实的期望和严格的界限。
5．管理你的时间和精力。你需要做对三件事情：睡眠、饮食和锻炼。

成功回归的4个关键

1．重新包装你的技能。
2．重新组织你的经验。
3．重新连接职业生态系统。
4．重新建立你的自信。

跨国工作

  • 国际化思维，职业生涯中的关键区别点
  • 做些有意思的事，不要纯粹为了生意而活
  • 在45岁之前不要关上任何一扇门

职场危机

应对职场危机需要知道的5件事

1．应对职场危机的第一步应该是清晰客观地认识问题。
2．有时候，职场危机可能是由他人对你的技能、目标或业绩的错误认识引起的。
3．在其他情况中，你的不足之处可能并不只是认知上的，而且是真实存在的。
4．有的职场危机是可以预见并规避的。
5．无论你的职场危机是不可预知的还是可以预料的，你都需要从逆境中快速恢复的方法

  • 重新组织你的经验
  • 重新包装老旧过时或有所欠缺的技能
  • 重新连接职业生态系统
  • 重新建立自信

遭遇职场危机时，请将骄傲放在一边

  • 不要担心别的事，享受当下就行了
  • 弄清你的使命，从事你愿意投入的事业
